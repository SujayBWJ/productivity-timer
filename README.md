# Productivity Timer

A Pomodoro-style productivity timer built with React as part of my React learning journey.

This project focuses on understanding React fundamentals by building a real application from scratch instead of following a tutorial. The goal is to learn how React works internally while implementing common features like timers, state management, and side effects.

## Features

- Start timer
- Pause timer
- Reset timer
- Custom timer duration
- Progress indicator
- Alarm sound when the timer ends
- Responsive UI

## Tech Stack

- React
- Vite
- JavaScript (ES6+)
- CSS

## React Concepts Practiced

- Components
- JSX
- State Management with `useState`
- Event Handling
- `useEffect`
- Dependency Arrays
- Cleanup Functions
- `setInterval` and `clearInterval`
- Component Re-rendering
- Component Lifecycle
- Conditional Rendering

## Installation

Clone the repository:

```bash
git clone https://github.com/<your-username>/productivity-timer.git
```

Navigate into the project directory:

```bash
cd productivity-timer
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
